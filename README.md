## 🚀 What is it?

A fun browser-based drum kit that lets you play different drum sounds using your keyboard.

Perfect for experimenting with JavaScript events, DOM manipulation, and audio playback.

---

## 🎹 How to Play

1. Clone or download this repository.
2. Open the project in VS Code.
3. Launch `index.html` using Live Server (or open it directly in your browser).
4. Press the highlighted keys on your keyboard.
5. Become a rockstar. 🤘

---

## ✨ Features

- 🥁 Interactive drum sounds
- ⌨️ Keyboard controls
- ⚡ Instant audio feedback
- 🎨 Visual key animations
- 📱 Lightweight and fast
